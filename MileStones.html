Roadmap

This roadmap includes the status of each phase by using colored list items:
  1. For already past the phase
  2. For current phase
  3. For a future phase

  1. Project Olympic

    Version 0, May, 2015

    Ethereum had a large collection of Proof-of-Concept's for their blockchain. Their last few prototypes, that were public
    came to be known as Project Olympic

    These prototypes were mainly made to stress-test the network and blockchain, there was a bug bounty of 25,000 Ether.

  2. Project Frontier

    Version 1, July, 2015

    Frontier is the first in a series of releases that punctuate the roadmap for the development of Ethereum. (Ethereum Wiki)

  3. Project Homestead

    Version 2, February, 2016

    Homestead is the hardfork updating EVM's some changes to the protocol and more

  4. Project Metropolis(v1)

    Version 3, October, 2017

    Metropolis part 1 is there to make preperations for private transactions, parallel transactions, and it extends the period to the "Ice Age"

  5. Project Metropolis(v2)

    Version 3.5, TBA

    Metropolis part 2 is there to make way for private transactions, and more

    Some of the added projects to Ethereum potifolio will be an WebAssembly Ethereum node or wallet

  6. Project Serenity

    Version 4, TBA

    Serenity is there for starting the Casper protocol and more
